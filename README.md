# Weather Diary (Дневник погоды)

## ФИО автора
Мурзаканов Мартин Скиндарович

## Описание программы
Desktop-приложение на Python для ведения дневника погоды.  
Пользователь может добавлять записи о погоде, просматривать их в таблице и применять фильтры.

## Функционал
- Добавление записи с полями:
  - Дата (`YYYY-MM-DD`)
  - Температура (число, может быть отрицательным)
  - Описание погоды (обязательное поле)
  - Осадки (Да/Нет)
- Отображение всех записей в таблице
- Фильтрация:
  - По дате (точное совпадение)
  - По температуре (выше заданного значения)
  - Комбинированная фильтрация по дате и температуре
- Сохранение данных в `data/data.json`
- Автоматическая загрузка данных при старте приложения
- Обработка отсутствующего и поврежденного JSON-файла
- Валидация пользовательского ввода с сообщениями об ошибках на русском языке

## Как запустить
1. Установите Python 3.10+.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Тесты
Запуск тестов:
```bash
pytest
```

## Примеры использования
1. Введите:
   - Дата: `2026-04-27`
   - Температура: `14.5`
   - Описание погоды: `Солнечно`
   - Осадки: выключено
2. Нажмите кнопку `Добавить запись`.
3. Для фильтрации:
   - Укажите дату, например `2026-04-27`, и/или
   - Укажите температуру, например `10`, чтобы показать записи с температурой выше 10.
4. Нажмите `Фильтровать`.
5. Нажмите `Сбросить фильтры`, чтобы снова увидеть все записи.
